# <font color=Blue size=10>前后端分离项目入门</font>
* 1、<font color=Blue size=10>技术栈</font>

| 前端                | 描述        |
|-------------------|-----------|
| Vue               | 前端框架      |
| VueX              | 全局状态管理框架  |
| ElementUI         | 前端UI框架    |
| Axios             | 前端HTTPS框架 |
| vue-element-admin | 项目脚手架     |


| 后端              | 描述       |
|-----------------|----------|
| SpringBoot      | 后端框架     |
| Mybatis         | ORM框架    |
| Redis           | 非关系型数据库  |
| lombok          | java类库   |
| fastjson2       | 解析json数据 |
| spring-security | 安全框架     |
